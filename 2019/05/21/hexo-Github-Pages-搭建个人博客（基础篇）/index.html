环境安装(node.js、git)

  • node.js(官网下载)是我们所要用到的hexo的实现基础，而所谓的hexo其实就是一个静态网页生成器，具体的细节等到后面再讲，这里先把它一步步安装到底即可。

  • git(Github镜像下载)是一个开源的分布式版本控制系统，它的角色就是一个更新博客内容时的中间人。不知道什么原因，国内访问官方服务器的速度堪忧，即使费力下载好，也会在后续使用其中各种npm指令做包管理的时候领受到加倍痛苦。故优选此淘宝镜像下载。

  • Hexo 是一个快速、简洁且高效的博客框架。它使用 Markdown（或其他渲染引擎）解析文章，以生成静态网页，并且可选的博客主题相当丰富，自定义与个性化的空间也很大。安装方法：在任意cmd、powershell或git bash中输入：

    $ cnpm install -g hexo-cli

    其中的美元符号是提示符无需输入，cnpm是从淘宝镜像安装的命令提示符，如果是从官方安装的git，可以运行以下命令安装cnpm以替代npm：

    $ npm install -g cnpm --registry=https://registry.npm.taobao.org
